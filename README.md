# Каталог кофе

## Технологии

- Svelte
- Webpack
- TypeScript
- Less
- Jest
- ESLint
- Prettier
- Husky

## Запуск

`yarn` - установка зависимостей

`yarn dev` - запуск dev-сервера  
`yarn build:dev` - сборка dev-версии  
`yarn build:prod` - сборка prod-версии  
`yarn test` - запуск тестов  
`yarn lint` - запуск линтера  
`yarn format` - форматирование кода
